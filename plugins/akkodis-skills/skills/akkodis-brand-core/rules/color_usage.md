# カラー使用ルール

出典：Akkodis Brand Guidelines V3 §5 / Brand Checklist §2

## ブランドパレット

正式値は `tokens/colors.json` を参照。

| 色名 | HEX | 主用途 |
|---|---|---|
| Akkodis 濃いブルー | `#001F33` | 背景・本文・ネガティブロゴ |
| Akkodis イエロー | `#FFB81C` | アクセント・グラデーション(2/3) |
| Akkodis 鋼青色 | `#00FFFF` | アクセント・グラデーション(1/3) |
| ブラック | `#000000` | テキスト・補助 |
| ホワイト | `#FFFFFF` | 背景・テキスト |

各色に20%/40%/60%/80%トーンが許容される。これら**5色＋4トーン以外**のカラーをブランド要素に使用してはならない（データビジュアライゼーションは別途データパレット可）。

## 色彩配分の原則

> **ベース：濃いブルー or ホワイト**　・　**アクセント：イエロー、鋼青色**

- 広範囲のイエロー・鋼青色を背景に使用してはならない（可読性低下）
- 濃いブルー or ホワイトのベースに、少量のイエロー・鋼青色アクセントを配する

## グラデーション規定

| 項目 | ルール |
|---|---|
| 比率 | イエロー2/3 ＋ 鋼青色1/3 |
| カラーストップ | 0% / 66% / 100% |
| 角度 | 45° |
| 鋼青色配置 | 必ず**右上**に |

**許可される使用箇所**：
- アイコン内
- アニメーション
- Akkodisスタンプ
- グラフ
- ソーシャルメディア投稿の背景

**禁止**：
- 本文テキストへの適用
- イメージフィルターとしての使用

**例外**：Web画面で判読性確保のためのオーバーレイは可だが、グローバルブランドチーム承認必須。

## アクセシビリティ配慮（CUD）

- **白×黄／黄×白の文字背景組み合わせは禁止**（特に視覚障害配慮文書）
- 一般文書でもこの組み合わせは非推奨
- アクセシビリティ配慮資料では BIZ UDPゴシック（日本語）／Arial（英語）を使用

## 印刷時の注意

鋼青色（`#00FFFF`）の鮮やかさを維持するには、可能であれば Pantone 3105C で印刷することを推奨。
