# ロゴ運用ルール

出典：Akkodis Brand Guidelines V3 §3 / Logo Guideline V1 / Brand Checklist §1

## カラーバリアント選択フロー

| 背景 | 推奨ロゴファイル |
|---|---|
| ホワイト/明るい単色背景 | `AKKODIS_Logo_RGB_BLUE.png`（プライマリー：ブルー/イエロー） |
| 濃いブルー/暗い背景 | `AKKODIS_Logo_POS_RGB.png`（ネガティブ：ホワイト/イエロー） |
| イエロー背景 | `AKKODIS_Logo_RGB_BLUE.png`（プライマリー：ブルー/ホワイト派生） |
| モノクロ印刷 | `AKKODIS_Logo_RGB_BLACK.png` |
| 暗色背景の単色用途 | `AKKODIS_Logo_RGB_WHITE.png`（モノクロ：ネガティブホワイト） |
| ネガティブ全般 | `AKKODIS_Logo_NEG_RGB.png` |

「単色/モノクロは難度が高い使用」のため、プライマリーが適切でない場合に限り使用する。

## 最小サイズ

| 種別 | 印刷 | 画面 | 適用条件 |
|---|---|---|---|
| タグライン付き | 40mm | 200px | これ未満ではタグラインなし版へ切り替え |
| タグラインなし | 10mm | 50px | これより小さい使用は禁止 |

## クリアスペース

ロゴ内の **「A」文字の幅** を1単位として、上下左右に1単位以上の余白を必ず確保する。
この範囲にテキスト・グラフィック・画像を配置してはならない。

## 改変禁止事項

以下は全て禁止。原本ファイルをそのまま使用すること。

- 縦横比を変更しての変形・歪曲
- 回転（必ず水平で使用）
- 色の変更（マーケティングチーム承認色のみ使用可）
- 構成要素の改変（例：「i」の点の形を変える）
- 他フォントの重ね合わせ
- ドロップシャドウ、発光、その他スタイル効果の追加
- 代替タグライン、部門名、商品名のロックアップ追加

## 例外運用

- 外部向けモーショングラフィックス最終フレームでのタグライン付き使用は可
- ロックアップが必要な戦略的場面では、グローバルブランドチームの承認を得ること

## ファイル配置

このSkillでは `assets/logos/` 配下に5種のロゴPNGを配置済み。全て寸法は1182×149px、アスペクト比約7.93:1。
