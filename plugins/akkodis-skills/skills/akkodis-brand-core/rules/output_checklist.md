# 成果物セルフチェックリスト

成果物（DOCX/PPTX/XLSX/画像）を生成・納品する直前の最終チェックリスト。
このチェックを通過しない限り提出してはならない。

## 共通チェック

- [ ] **ロゴ**：`assets/logos/` の原本のみ使用。改変・回転・色変更・効果追加なし
- [ ] **ロゴ最小サイズ**：印刷40mm/画面200px（タグライン付き）または10mm/50px（タグラインなし）以上
- [ ] **クリアスペース**：ロゴ周囲にAサイズ分の余白を確保
- [ ] **カラー**：`#001F33` / `#FFB81C` / `#00FFFF` / `#000000` / `#FFFFFF` ＋4トーン以外を使っていない
- [ ] **ベース色**：濃いブルー or ホワイトをベースとし、イエロー/鋼青色は**アクセント限定**
- [ ] **白×黄／黄×白の組み合わせ**を文字背景に使用していない
- [ ] **グラデーション**：イエロー2/3＋鋼青色1/3、45°、鋼青色右上配置を遵守
- [ ] **本文テキストへのグラデーション適用なし**
- [ ] **フォント**：Tier 1（プライマリー）または Tier 2（Office デフォルト）から選択
- [ ] **日本語ビジネス文書**は Meiryo UI を使用
- [ ] **MS明朝・HGフォント・游ゴシック**をビジネス文書で使っていない
- [ ] **日本語フォントで英語テキストを記載**していない

## 表記チェック

- [ ] **AKKODiS**（小文字 i）：日本国内文脈で `Akkodis` `AKKODIS` と表記していない
- [ ] **正式社名**：AKKODiSコンサルティング株式会社／AKKODiS Consulting Ltd.
- [ ] **サービス順序**：Consulting → Solution → Academy → Talent
- [ ] **Solution**（単数形）：`Solutions` と書いていない
- [ ] **AKKODiS innovation Lab.**（i は小文字）
- [ ] **GDC初出**：`Global Delivery Center（GDC）` と正式名称併記
- [ ] **テックコンサルタント**：`エンジニア社員` `無期雇用派遣エンジニア` と書いていない
- [ ] **無期雇用派遣**：`特定派遣` と書いていない（改正派遣法）
- [ ] **就業**：`稼働` と書いていない
- [ ] **キャリアコーチ／キャリアプランナー**：英語表記していない
- [ ] **ビジョン**：「日本企業を、世界企業へ、現場変革から。」を使用
- [ ] **人財／人材**：原則「財」、Web SEO・公的文書のみ「材」

## IOWN®表記チェック（最重要）

- [ ] **IOWN®（単体）**：®省略マーカーが無い文書、およびマーカーより前の単体IOWNは ® 必須
- [ ] **®省略マーカー**：`IOWN®（Innovative Optical and Wireless Networkの略で、NTT株式会社の商標または登録商標。以下、IOWN）` を任意の単体IOWN1か所に付した場合、**その位置以降**の単体IOWNは ® 省略可（初出非依存・I-10）
- [ ] **初出時**：`IOWN®（Innovative Optical and Wireless Network）` で正式名称併記
- [ ] **IOWN GLOBAL FORUM™**：全大文字＋™、`IOWN Global Forum` と書いていない
- [ ] **AKKODiSの位置付け**：`スポンサーメンバー`（「メンバー」のみは不可）
- [ ] **付属語あり**（IOWN構想／IOWN推進室／IOWN構想基礎研修）：®を**付けない**
- [ ] **主幹説明**：「IOWN GLOBAL FORUM™が中心」（NTT中心と書いていない）
- [ ] **注釈**：固定文を使用、独自注釈なし

## ファイル形式別の追加チェック

### PPTX
- [ ] スライドサイズ：16:9 ワイドスクリーン（13.33×7.5インチ）
- [ ] テーマカラー：「Akkodis 2」または「Akkodis 3」スキーム使用
- [ ] テーマフォント：Akkodis（Meiryo UI）スキーム使用
- [ ] 公式テンプレート `assets/templates/akkodis_pptx_template_dark.pptx` のレイアウトを優先使用

### DOCX
- [ ] 用紙：A4
- [ ] 余白：上下左右 25.4mm（1インチ）以上
- [ ] フォント：日本語 Meiryo UI、英語 Arial（または Tier 1 のRoboto/Tiempos）
- [ ] 本文色：基本は黒または濃いブルー `#001F33`

### XLSX
- [ ] ヘッダー行：濃いブルー `#001F33` 背景＋ホワイト文字、または逆配色
- [ ] アクセントセル：イエロー `#FFB81C` を限定的に使用
- [ ] グラフ配色：ブランドパレット5色＋トーン内で構成

### 画像／SVG
- [ ] ロゴ配置時のクリアスペース確保
- [ ] グラデーション規定遵守（45°、2/3:1/3、右上鋼青色）
- [ ] フォトグラフィー規定遵守（人物：自然な笑顔、目の高さ、自然光、中程度の被写界深度）

## アクセシビリティ

- [ ] 視覚障害者向け／公共資料の場合：BIZ UDPゴシック使用
- [ ] コントラスト比 WCAG AA 以上（特に本文）
- [ ] 白×黄組み合わせ未使用

## テキスト校正

漢字開き／送り仮名／敬語等のテキスト校正は本Skillの責任範囲外。
姉妹Skill `akkodis-proofreader` を併用すること。
